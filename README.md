# Rick and Morty Characters

Web-app to watch chars from Rick and Morty show.

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Rick and Morty API

## Functions

- See all 826+ cahrs form R&M API
- Searck names with debounce
- Adaptive design
- Char status (alive/dead/unkown)

## install

```bash
git clone ...
cd "repo name"
npm install
npm run dev
```
